# Plan: robots.txt と sitemap.xml の追加 (#579)

## Context

dowo.dev に `robots.txt` と `sitemap.xml` が存在しない。クローラーへの指示とサイトマップの提供ができておらず、新規記事のインデックス登録が遅れている。

## 実装内容

### 1. `public/robots.txt` を新規作成

完全に静的な内容のため `public/` に直接配置する。

```
User-agent: *
Allow: /
Sitemap: https://dowo.dev/sitemap.xml
```

### 2. `src/app/sitemap.ts` を新規作成

ビルド時に Hasura から記事一覧を取得し、サイトマップを動的生成する。既存の `generateStaticParams`（`src/app/blog/[slug]/page.tsx:15-22`）と同じパターンを再利用。

- `initializeApollo()` + `GET_ARTICLES_BY_STATUS` で公開記事を取得
- トップページ（`/`）+ 全公開記事（`/blog/{slug}`）の URL を含める
- `lastModified` に `updated_at` を使用

**再利用する既存コード:**
- `src/lib/apolloClient.ts` → `initializeApollo()`
- `src/queries/queries.ts` → `GET_ARTICLES_BY_STATUS`
- `src/gql/graphql.ts` → `GetArticlesByStatusQuery`

## 検証

```bash
npm run build
```

- `out/robots.txt` が生成されていること
- `out/sitemap.xml` が生成され、全公開記事の URL が含まれていること
